# 🛡 EcoGuard — Oil Spill Detection System
### Real-time monitoring & alert system for Petroleum Engineering students

---

## Features
- **Live Sensor Dashboard** — monitors 12+ sensor nodes in real-time
- **Auto-Detection Engine** — scans every 3 seconds, detects threshold breaches
- **Instant Alerts** — visual flash, toast notifications, push notifications
- **Interactive Map** — shows sensor locations and spill zones
- **Incident Reports** — export CSV reports for academic use
- **Offline-capable PWA** — works without internet after first load
- **Installable** — can be added to home screen on Android/iOS/Desktop
